# Canva SVG Editor Plugin

Canva üzerinde çalışan SVG düzenleme plugin'i. Seçilen SVG'leri parçalarına ayırma, gruplama ve renk değiştirme özellikleri sunar.

## Özellikler

- **SVG Parçalama (Ctrl+U)**: SVG'yi path ve group elementlerine ayırır
- **SVG Gruplama (Ctrl+G)**: Seçili elementleri gruplar
- **Renk Değiştirme**: Her bir çizimin rengini değiştirme (fill ve stroke)

## Kurulum

```bash
npm install
```

## Geliştirme

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Kullanım

1. Canva'da bir SVG seçin
2. Ctrl+U ile parçalarına ayırın
3. Ctrl+G ile gruplayın
4. Renk değiştirmek için color picker'ı kullanın
